This plugin enables different payment methods associates with pay4later for woocommerce

== Description ==
This plugin adds pay4later payment methods for Woocommmece.

If you set the amount of money that needs to be spent to &pound;500 then unless the basket total is equal to that amount or higher then pay4later finance will not be available for checkout.


== Installation Guide ==

1. Upload the plugin to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to WooCommerce > Setting > Checkout then configure the payment methods by enabling each methods.
